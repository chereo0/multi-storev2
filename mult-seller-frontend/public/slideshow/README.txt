Place your slideshow images here.

Option A — Simple naming (no JSON needed):
- Use any of these names (first found, up to 8 used):
- sd1.(jpg|jpeg|png|webp)
- sd2.(jpg|jpeg|png|webp)
- sd3.(jpg|jpeg|png|webp)
- sd4.(jpg|jpeg|png|webp)
- sd5..sd8 are also supported.
- Alternates also supported: slide1..slide4, hero-1..hero-4, banner-1..banner-4

Option B — Manifest (flexible filenames):
- Create a JSON file here: index.json
- Content example:
	[
		"my-first-banner.webp",
		"neon-checkout.png",
		"explore-choices.jpg"
	]
- Files should be wide, at least 1600px width for crisp results on large screens.
- Use any of: .jpg .jpeg .png .webp (case-insensitive)
- After copying, refresh the app to see them.
